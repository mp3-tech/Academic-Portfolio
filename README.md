# 知識管理期末實務：電腦繪圖創作之知識體系建模 (評分：98/100)

## 專案核心
本專案透過對數位繪圖專家的深度訪談，將感性的藝術創作過程轉化為理性的知識管理流程。展示了將「資訊組織」專業應用於跨領域專案管理的能力。

## 技術成果
* **WBS (知識分解結構)**：將繪圖流程嚴謹拆解為「前製準備、結構開發、色彩處理、後製輸出、資產保存」五大核心階段。
* **PERT (知識執行程序)**：定義任務間的 FS/SS/FF/SF 相依關係，透過分流點與合流點分析，優化並行作業效率。
* **RAM (責任指派矩陣)**：導入 **PRASI 模型** 進行權責劃分，確保「執行者(P)、審核者(R)、核准者(A)、支援者(S)、被告知者(I)」在跨領域溝通中資訊對等。

## 關鍵洞察
這份報告證實了知識管理並非單純的理論，而是能將隱性知識（靈感與美感）結構化為顯性流程（SOP）的強大工具，此邏輯亦延伸應用於我目前開發的 AI 自動化工作流。
